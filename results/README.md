# Evaluation Results

This folder contains evaluation results generated by the batch evaluation script.

## Generating Results

```bash
# Save results to this folder
python scripts/asr_batch_evaluate.py --output results/kotoba_evaluation.json

# With verbose output
python scripts/asr_batch_evaluate.py --verbose --output results/assemblyai_evaluation.json --model assemblyai
```

## Result File Format

Each JSON file contains metadata, per-sample results, and aggregate statistics including CER, WER, RTF, and processing times.

## Notes

- Result files are git-ignored by default
- Use descriptive filenames for different experiments
